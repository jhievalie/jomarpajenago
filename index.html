E-commerce Sales & Marketing Dashboard

Power BI · DAX · SQL · Excel · Data Modeling

Problem

Sales and marketing data were spread across multiple sources (Excel, SQL databases, and platform exports), making it difficult to track performance consistently. Reporting required manual consolidation, which was time-consuming and prone to inconsistencies. There was no unified view of KPIs such as revenue, conversion rate, and channel performance, limiting the ability to make timely, data-driven decisions.

Process

I designed and developed an end-to-end BI solution using Power BI to centralize and automate reporting:

  • Consolidated raw data from multiple sources using SQL and Excel
  • Cleaned and transformed datasets to ensure consistency and accuracy
  • Built a structured data model optimized for performance and scalability
  • Created DAX measures for key KPIs (Revenue, AOV, CVR, ROAS)
  • Designed an interactive dashboard with dynamic filters (MTD, YTD, All Time)
  • Implemented funnel and channel performance analysis for deeper insights

The dashboard was built with a focus on clarity, usability, and decision-making speed.

Result

  • Reduced reporting time by up to 40% through automation
  • Enabled real-time monitoring of sales and marketing performance
  • Improved visibility into customer behavior and conversion funnel
  • Allowed faster identification of high-performing channels and campaigns

This solution transformed fragmented data into a centralized, decision-ready dashboard.
